Manual: How to make a Sample

1. Obligatory Filters

  • Name a Sample enables you to name a sample that you are going to work with (e.g. «Sample for my MA thesis»).
  • Sampling Algorithm enables you to choose the sampling method. See more about available sampling methods on the About page.
  • Sample Size enables you to choose the size of your future sample. Keep in mind that you are limited by the existing number of languages as well as the number of languages with available descriptions.
  • Macroareas enables you to choose languages from which macroareas you want to include in your sample. The macroarea list is as in Glottolog.
  • Descriptions' Languages enables you to choose the languages of the descriptions. For example, if you are able to read only in English, the algorithm will suggest to you only the languages with English descriptions available.

2. Extra Filters

  • Include Languages enables you to include in the sample particular languages that you would like to work with.
  • Exclude Languages enables you to exclude in the sample particular languages that you would not like to work with.
  • Add a Grambank feature enables you to restrict the sample to languages that have particular characteristics in the Grambank database. See the Grambank site to explore the available parameters. Be aware that the number of languages annotated in Grambank is limited.
  • Add a WALS feature enables you to restrict the sample to languages that have particular characteristics in the WALS database. See the WALS site to explore the available parameters. Be aware that the number of languages annotated in WALS is limited.

3. Search Settings

  • Select Languages By enables you to choose, according to which parameters the languages are chosen by the algorithm. In order to make results less deterministic, we have added random noise in each ranking method.
    • Random: No specific filter is set. The languages are randomly ranked.
    • Extensiveness of description (2 · pages + 0.5 · year): The formula given here is our approximate estimation of description degree. The formula applies to scaled year and pages values.
    • Total number of Descriptions: The languages with the highest number of descriptions will be chosen.
    • Descriptions' publication year: The languages with the most recently published descriptions will be chosen.
    • Descriptions' page count: The languages with the highest number of descriptions' pages (in total, all descriptions counted) will be chosen.
  • Preference for Descriptions enables you to select the type of descriptions you're interested in as this preference modifies any ranking method listed above. For instance, if your research focuses on grammatical number, you will probably want to explore grammatical descriptions. On the other hand, if you are studying verbal colexifications, you may prefer languages that have dictionaries available. Positive values favor languages with more grammars, negative values favor languages with more dictionaries. This feature works by adjusting the ranking score based on the grammar/dictionary ratio in the language's sources.
